# 肩键如何设置

<img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/V0FM/zh-CN/images/icon-note.svg" width="24" height="24"/>当前功能仅部分游戏支持。

## **设置步骤**

1.  启动游戏后，单击悬浮手柄的功能键可快速进入肩键配置界面。
2.  长按+拖动界面LB/RB虚拟键位图标，至游戏中需要映射的按钮区域（如：技能键/加速键）
3.  点击界面上方保存按钮，完成映射配置。

## **示例说明**（QQ飞车）

1.  单击悬浮手柄功能键<img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/V0FM/zh-CN/zh-cn_image_0000002483849749.png" width="24" height="24"/>。
    
    ![](https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/V0FM/zh-CN/zh-cn_image_0000002450490292.png)
    
2.  进入肩键映射配置界面。
    
    ![](https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/V0FM/zh-CN/zh-cn_image_0000002450649912.png)
    
3.  长按拖动“RB” 虚拟键位图标，至“小喷加速”按钮上方。
    
    ![](https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/V0FM/zh-CN/zh-cn_image_0000002483853909.png)
    
4.  长按+拖动“LB” 虚拟键位图标，至“氮气加速”按钮上方。
    
    ![](https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/V0FM/zh-CN/zh-cn_image_0000002450654442.png)
    
5.  点击界面上方“保存”按钮，完成映射配置。
    
    ![](https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/V0FM/zh-CN/zh-cn_image_0000002450814066.png)
    
6.  比赛中按压手柄顶部L/R键即可触发赛车加速效果。
  
    
    <img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/V0FM/zh-CN/images/icon-note.svg" width="24" height="24"/>当前版本LB/RB映射功能默认开启，暂不支持隐藏或关闭（该功能将在后续系统升级中开放）。
