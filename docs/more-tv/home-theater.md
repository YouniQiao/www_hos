# 分布式家庭影院

华为智慧屏与音箱组建分布式家庭影院后，在播放视频或音乐时，智慧屏与音箱同时发声，享受沉浸式视听体验。

1.  将华为智慧屏和智能音箱连接至同一WLAN，开启智慧屏和音箱蓝牙。
2.  智慧屏会扫描周围存在的音箱，在界面上会弹出创建无线家庭影院系统界面，按照界面提示完成组网即可。
    
    ![](https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/V0FM/zh-CN/zh-cn_image_0000002332288925.png)
    
    如果创建无线家庭影院系统弹窗弹出时，如未进行配置，后续可按如下步骤配置：
    
    1.  在智慧屏桌面点击设置 > 声音 > 输出，点击选择WLAN音箱。
    2.  打开自动发现音箱开关，弹框进入组网模式选项。
    3.  在弹框中选择模式，点击确定。
    
3.  请根据屏幕提示，按照音箱的实际摆放位置和数量进行配置，即可完成家庭影院连接。
