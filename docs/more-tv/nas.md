# 华为智慧屏如何访问NAS设备

华为智慧屏支持连接NAS（Network Attached Storage 即网络存储）设备，可以使用文件管理应用查看NAS设备的图片和视频。

<img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/V0FM/zh-CN/images/icon-note.svg" width="24" height="24"/>需确保智慧屏与NAS设备处于同一局域网。

## 发现NAS设备

在智慧屏桌面打开文件管理<img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/V0FM/zh-CN/zh-cn_image_0000002475277852.png" width="24" height="24"/>，点击网络邻居即可自动发现NAS设备。

![](https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/V0FM/zh-CN/zh-cn_image_0000002477079724.png)

## 连接NAS设备

点击发现的NAS设备，在弹出的输入框，输入共享设备用户名和密码。

<img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/V0FM/zh-CN/images/icon-note.svg" width="24" height="24"/>

+   用户名和密码需要在NAS设备侧设置，若NAS设备支持匿名登录可直接勾选匿名登录不需要输入用户名密码。
+   用户名和密码设置有疑问请联系NAS设备品牌方询问。

![](https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/V0FM/zh-CN/zh-cn_image_0000002508999699.png)


<img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/V0FM/zh-CN/images/icon-note.svg" width="24" height="24"/>NAS设备第一次和智慧屏连接后，若出现NAS重启的情况，需要重新连接智慧屏。

+   NAS设备显示灰色状态表示未和智慧屏连接。
+   NAS设备显示绿色状态表示未和智慧屏连接。

## 访问NAS设备

输入用户名密码即可登录成功，访问NAS设备中的资源。

![](https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/V0FM/zh-CN/zh-cn_image_0000002508999761.png)
