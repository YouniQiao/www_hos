# 与iOS手机配对连接

> 本文适用于 **WATCH Ultimate 非凡大师、WATCH GT 6 系列和WATCH 5**。

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="ultimate" label="WATCH Ultimate">



1. 安装

<img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000002374480977.png" width="24" />

。如已安装，请升级至最新版本。 如未安装，请到手机App Store 搜索，下载并安装。
2. 进入，点击 > 。如果没有华为帐号，请先注册后再登录。 > 💡 **提示：** 为保证您的穿戴设备与手机配对连接成功，建议您使用成人华为帐号登录。
3. 穿戴设备开机，选择语言，将设备靠近手机，在首页，点击上角 > ，在扫描结果中找到您的设备，点击。也可以点击，直接扫描设备上的二维码。
4. 当穿戴设备屏幕出现配对请求时，点击 <img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000002340602994.png" width="24" /> 
。手机上界面出现蓝牙配对请求提示框，点击。手机出现是否允许设备显示手机通知的提示框，请点击，实际请按界面指引进行操作。
5. 等待几秒钟，提示与穿戴设备配对成功。
> 💡 **提示：** 请不要在手机上手动上滑关闭，以防穿戴设备与断连，无法接收手机消息和来电。 为确保穿戴设备能及时收到通知信息，请在手机上确认消息通知设置是否正常。

</TabItem>
