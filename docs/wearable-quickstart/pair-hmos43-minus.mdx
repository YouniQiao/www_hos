# 与HarmonyOS 4.3及以下版本的手机配对连接

> 本文适用于 **WATCH Ultimate 非凡大师、WATCH GT 6 系列和WATCH 5**。

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="ultimate" label="WATCH Ultimate">



1. 安装

<img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000002374600597.png" width="24" />

。如已安装，请打开，点击 > ，升级至最新版本。
2. 进入，点击 > 。如果没有华为帐号，请先注册后再登录。 > 💡 **提示：** 为保证您的穿戴设备与手机配对连接成功，建议您使用成人华为帐号登录。
3. 穿戴设备开机，选择语言后准备配对。
4. 将穿戴设备靠近手机，进入首页，点击界面上角 > ，在扫描结果中找到您的穿戴设备，点击。也可以点击，直接扫描穿戴设备界面展示的二维码。
5. 界面出现，点击。
6. 当穿戴设备屏幕出现配对请求时，点击 <img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000002374480733.png" width="24" /> 
，同时在手机端根据界面提示完成穿戴设备和手机配对。 > 💡 **提示：** 为了确保穿戴设备能及时收到通知信息，请在手机上确认消息通知设置是否正常。

</TabItem>
