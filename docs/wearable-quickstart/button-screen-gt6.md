# 按键及屏幕操作

> 本文适用于 **WATCH GT 6 系列**。

新机体验


|  | 按键及屏幕操作 |
| --- | --- |


   按键及屏幕操作

    手表采用彩色屏幕，支持全屏触摸、上滑、下滑、左滑、右滑、长按操作。


## 上键


> 💡 **提示：** 来电、锻炼等过程中有特别定义的场景例外。


| 操作 | 功能 |
| --- | --- |
| 旋转 | 在星环桌面或列表桌面，上下滑动应用。在星环桌面首界面，逆时针旋转表冠或下拉屏幕可显示/隐藏应用名称。 进入具体应用后，上下滚动屏幕。 星环桌面和列表桌面切换方式： 在星环桌面或列表桌面，上滑屏幕至应用最后一页，点击**星环桌面**或**列表桌面**切换。 点击**设置 > 表盘和桌面** > **桌面**，选择**星环**或**列表**模式。 |
| 单击 | 手表熄屏时，单击上键点亮屏幕。 手表显示表盘界面时，单击上键进入应用界面。 手表显示非表盘界面时，单击上键返回表盘界面。 蓝牙来电时，单击上键可将手表上的来电静音。 |
| 双击 | 手表亮屏状态下，双击上键查看所有运行中的应用。 例如，锻炼运行时，双击上键进入后台运行任务，点击表盘卡片回到表盘界面。 |
| 长按 | 手表关机状态时，长按上键开机。 手表开机状态时，长按上键，出现重启/关机选项画面。 手表开机状态时，长按上键 12s 以上，可强制重启。 |


## 下键


| 操作 | 功能 |
| --- | --- |
| 单击 | 单击下键可直接进入快捷功能页面。 下键自定义： 单击手表上键，在应用中选择**设置 > 按键设置** > **单击下按键**。 选择目标应用，完成手表下键自定义。设置完成后，在表盘界面单击手表下键，可进入快捷功能页面。 |
| 双击 | 打开钱包应用。 |
| 长按 | 调起**智慧语音**。 |


## 组合键


| 操作 | 功能 |
| --- | --- |
| 同时单击上键和下键 | 截屏，图片上传到手机图库。 |


## 屏幕基本手势


| 操作 | 功能 |
| --- | --- |
| 单击 | 选择确认。 |
| 长按 | 设备在表盘界面时，长按可快速更换表盘。 设备在卡片界面时，长按显示卡片管理界面。 |
| 上滑 | 查看消息通知（在表盘界面上滑）。 |
| 下滑 | 进入控制中心（在表盘界面下滑）。 可开启**找手机**、设置**情景模式**等。 可查看剩余电量和蓝牙连接状态。 |
| 向左滑 | 表盘界面向左滑查看手表卡片。 |
| 向右滑 | 表盘界面向右滑进入负一屏。 应用子页面向右滑返回上一级。 |
