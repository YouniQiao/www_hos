# 双模式切换

> 本文适用于 **WATCH Ultimate 非凡大师和WATCH 5**。


手表支持切换全能模式和省电模式。全能模式下，手表拥有完整的体验；在省电模式下，可提升续航时间，但是部分功能不可用，如移动网络、视频表盘等。
## 开启省电模式
全能模式下，在表盘界面从顶部下滑进入，点击，退出全能模式，进入省电模式。
## 开启全能模式
省电模式下，在表盘界面从顶部下滑进入，点击，退出省电模式，进入全能模式。
> 💡 **提示：** 当您的手表电量低于最低值时，会自动切换到省电模式；充电至电量高于最低值时，也会自动切换到全能模式。
