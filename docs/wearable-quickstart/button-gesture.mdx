# 按键及屏幕手势操作

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="ultimate" label="WATCH Ultimate">


手表采用彩色屏幕，支持全屏触摸、上滑、下滑、左滑、右滑、长按操作。



## 智感窗




| 操作 | 功能 |
| --- | --- |
| 轻压 | 佩戴手表状态下，在表盘界面，轻压智感窗3s后可快捷启动微体检功能等。
在表盘界面（仅部分表盘支持，如**山水墨境**），点击表盘进入表盘交互状态后按压智感窗，按压力度不同则表盘变化效果不同。 |
| 轻触 | 高血压风险评估、ECG心电分析、指尖血氧测量时，需要轻触智感窗。 |

</TabItem>
