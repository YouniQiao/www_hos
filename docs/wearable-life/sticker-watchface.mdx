# 设置趣味贴纸表盘

> 本文适用于 **WATCH Ultimate 非凡大师和WATCH GT 6 系列**。

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="ultimate" label="WATCH Ultimate">


您可选择或上传多张贴纸，配合多种个性化样式及功能控件，打造您的个性化表盘。还可点击屏幕与贴纸互动，获得好玩的惊喜体验。

1. 进入**华为运动健康**的设备详情页，点击**我的表盘**，选择**趣味贴纸**。
2. 进入**几何**、**心情**、**萌宠**等页面，选择喜欢的贴纸，还可点击**自定义 > <img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000002219835673.png" width="24" />**上传贴纸图片。
3. 选择喜欢的**布局**、**背景颜色**、**时间字体**等，还可选择**功能**，更换表盘中显示的数据。
4. 点击**设为当前表盘**，设备表盘将自动切换为所设置表盘，点击屏幕可与贴纸互动。

</TabItem>
