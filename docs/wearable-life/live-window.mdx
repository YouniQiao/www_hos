# 实况窗管理

> 本文适用于 **WATCH Ultimate 非凡大师、WATCH GT 6 系列和WATCH 5**。

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

   实况窗管理

    手表实况窗可以同步显示华为手机的实况窗信息，为您提供贴心的实况提醒服务，随时掌握当前任务的实时信息。


> 💡 **提示：** 使用实况窗功能，请确保手表已升级至最新版本。 实况窗功能仅支持配对 HarmonyOS 4.0 及以上版本的华为手机时使用，不支持三方安卓及iOS手机。


## 开启实况窗


1. 在穿戴设备应用列表中，点击**设置 > 表盘和桌面** > **实况窗**，开启实况窗开关。
2. 在手机**华为运动健康**的设备详情页，点击**通用设置 > 实况窗**， 打开**手机实况窗信息同步至手表实况窗**开关。


## 删除实况窗消息


- 手机锁屏界面删除对应实况消息，手表侧实况消息同步删除。
- 手机通知栏删除对应实况消息，手表侧实况消息同步删除。
- 点击手机实况窗胶囊右滑删除，手表侧实况消息同步删除。
- 点击手表实况窗实况信息，进入实况窗消息列表左滑，点击删除按纽，可以删除手表的实况消息。


## 更多操作


- 在手表表盘界面上滑进入消息中心，点击实况卡片可进入实况通知详情页，查看对应实况通知详情。
- 在手表表盘界面上滑进入消息中心，点击实况堆叠卡片可进入实况通知列表，查看当前所有实况窗卡片。
- 在手表表盘界面的屏幕下半屏，向下滑动可闭合实况窗。表盘实况窗手动下滑闭合隐藏后，再次在表盘界面上滑进入消息中心后，返回表盘实况窗可再次弹出。
