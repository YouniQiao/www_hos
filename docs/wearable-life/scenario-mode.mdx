# 设置情景模式

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="ultimate" label="WATCH Ultimate">


您可根据需要，在穿戴设备的控制中心快速开启/关闭免打扰、睡眠模式等。



## 开启免打扰


开启免打扰后，来电和通知静音且不振动（**闹钟**除外）。


立即开启：

1. 在设备表盘页面，从顶部下滑进入**控制中心**，点击**情景模式**，选择**免打扰**。
2. 设置后，**控制中心**会显示为**免打扰**。


如需关闭，请进入**控制中心**，点击**<img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000002194646884.png" width="24" />**图标可关闭免打扰模式。长按可更换模式或进行其他配置。


定时开启：

1. 您可从以下任一方式进入**免打扰**设置页面： 在设备表盘页面，从顶部下滑进入**控制中心**，点击**情景模式**，然后点击**免打扰**旁边的**<img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000002379196064.png" width="24" />**。
2. 进入设备应用列表，点击**设置 > 情景模式** > **免打扰**。
3. 点击**添加定时**，设置**开始时间**、**结束时间**、**重复日期**。


如需关闭定时开启，关闭对应时间段旁边的开关即可。

</TabItem>
