# 萌趣小胖手，猜拳赢手饰盲盒

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/figure/10044753_f010_fathands.png" alt="萌趣小胖手" width="560" height="" />

萌趣的小胖手主题，能猜拳能变装，动动小手能卖萌，装扮小胖手每天不重样。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/EMUI/C00B030/resource/card/202503041becsx/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1. 打开**主题**应用，在搜索框输入“小胖手”，进入小胖手主题后，点击**应用**。
2. 点亮屏幕，在小胖手锁屏界面，点击 **GAME** 进入猜拳游戏。
3. 点击下方的手势按钮出拳，连赢 3 局即奖励手饰，可直接佩戴。
4. 您可点击 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_fatdress2.png" width="24" height="24"/>，查看已赢得的手饰套装并更换。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/EMUI/C00B030/resource/card/202508300vZjQz/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

如您使用人脸解锁，为了更好的互动体验，请进入**设置 > 生物识别和密码 > 人脸识别 > 解锁屏幕**，建议使用**滑动解锁**。
