# 锁屏艺术签名，创作个性壁纸

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/figure/10044758_f013_Dynamic.png" alt="锁屏艺术签名" width="560" height="" />

使用艺术签名，体验签名文字与壁纸图片的艺术融合，还可通过 AI 创作，将喜欢的字体样式应用到签名中，让您的锁屏独具一格、个性十足。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/EMUI/C00B030/resource/card/202503041becsx/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1. 进入**图库**，选择图片，点击 **<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_more.png" width="24" height="24"/> > 设置为壁纸**。
2. 点击时钟区域，选择**艺术签名**。
3. 点击艺术签名区域，进行个性化设置：
    * 点击**艺术签名**下方的文本框，输入签名文本，6 个中文以内效果最佳。
    * 点击 **<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/textformat_size_square_fill-01.png" width="24" height="24"/>** 更换字体样式，还可点击 **<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_ai_font_12-01.png" width="24" height="24"/>** 选择带字的图片，AI 创作字体后一键应用到签名。
    * 点击 **<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_desktop_color-01.png" width="24" height="24"/>** 更换字体颜色。
4. 点击**应用 > 确定**，完成设置。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/EMUI/C00B030/resource/card/202508300vZjQz/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

建议选择主体突出、背景纯净的图片，以生成更好效果的艺术签名壁纸。
