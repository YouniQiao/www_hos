# 备忘录速记，速览摘要和原文

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/figure/20014428_f001_QuickNote.png" alt="备忘录速记" width="560" height="" />

浏览应用内容时，使用速记功能把链接添加至备忘录，不仅可自动生成摘要，还能一键快速复访原文，碎片化信息轻松整合。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/EMUI/C00B030/resource/card/202503041becsx/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

**添加速记快捷开关：**

1. 首次使用时，从屏幕顶部右侧下滑出**控制中心**，点击 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_controlpanel_edit.png" width="24" height="24"/>，在快捷开关面板中点击第二个 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_quickswitch_add.png" width="24" height="24"/>。
2. 点击**速记**上的 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_quickswitch_add1.png" width="24" height="24"/> 将其添加至面板，点击上方灰色区域退出添加界面，点击 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_choose_done.png" width="24" height="24"/> 保存设置。

**使用速记功能：**

1. 为了获得更佳体验，使用前请登录**华为账号**。
2. 打开要记录的内容，从屏幕顶部右侧下滑出**控制中心**，点击**速记**。
3. 点击 **添加链接** > <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_choose_done.png" width="24" height="24"/>，将原文链接保存至备忘录。
4. 点击屏幕顶部通知进入**备忘录**，打开对应备忘，链接下方会自动生成摘要；点击链接，可快速返回原文。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/EMUI/C00B030/resource/card/202508300vZjQz/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

部分应用不支持添加链接，部分敏感信息可能不支持生成摘要，请以实际体验为准。
