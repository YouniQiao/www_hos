# 元气心情主题，锁屏奇趣解压

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/figure/10044753_f011_MoodMatch.png" alt="元气心情主题" width="560" height="" />

设置元气心情主题，在锁屏界面体验与 3D 表情的趣味互动，点一点、按一按，好玩又解压。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/EMUI/C00B030/resource/card/202503041becsx/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1. 进入**设置** > **桌面和个性化**，点击**更多主题**，在**心情盒子**中选择**元气心情**。
2. 点击**自定义**进入编辑界面，可更换**背景样式**和**画面风格**。
3. 点击**应用**，体验元气心情主题，在锁屏界面点亮屏幕：
   - 点亮屏幕时，系统自动变换趣味动画。
   - 点击或长按表情，可触发趣味互动。
   - 充电时有专属充电动画，且在系统**响铃**模式下会伴随充电音效。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/EMUI/C00B030/resource/card/202508300vZjQz/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

- 首次使用该主题时，点击**去应用**进入主题编辑界面。
- 您可通过调节系统铃声音量，调节该主题的互动音效音量。
