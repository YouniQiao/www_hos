# 备忘录速记，速览摘要和原文

<iframe src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202504201ngpds/zh-cn/image/video/20014428_f001_QuickNote.mp4#toolbar=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="360" height="486"> </iframe>

浏览应用或网页时，使用速记功能把链接添加至备忘录，不仅可自动生成摘要，还能一键快速复访原文，碎片化信息轻松整合。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1.  为了获得更佳体验，使用前请登录华为账号。
2.  打开要记录的内容，从屏幕顶部右侧下滑出控制中心，点击速记。
3.  点击添加链接 > <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202504201ngpds/zh-cn/image/common/buttons/ic_choose_done.png" width="24" height="24"/> ，将原文链接保存至备忘录。
4.  进入备忘录，打开对应备忘，点击<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202504201ngpds/zh-cn/image/common/buttons/ic_public_arrow_down.png" width="24" height="24"/>，可查看全部摘要；点击链接，可快速返回原文。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

+   目前已支持华为浏览器等应用，部分敏感信息可能不支持生成摘要，请以实际体验为准。
+   首次使用速记功能时，需从屏幕顶部右侧下滑出控制中心，点击 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202504201ngpds/zh-cn/image/common/buttons/ic_controlpanel_edit.png" width="24" height="24"/> > <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202504201ngpds/zh-cn/image/common/buttons/ic_quickswitch_add.png" width="24" height="24"/> 展开快捷开关面板，点击速记上的 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202504201ngpds/zh-cn/image/common/buttons/ic_quickswitch_add1.png" width="24" height="24"/> > <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202504201ngpds/zh-cn/image/common/buttons/ic_choose_done.png" width="24" height="24"/> 将其添加到控制中心。
