# 锁屏艺术签名，创作个性壁纸

<iframe src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202504201ngpds/zh-cn/image/video/10044758_f013_Dynamic.mp4#toolbar=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="360" height="486"> </iframe>

使用艺术签名，体验签名文字与壁纸图片的艺术融合，还可通过 AI 创作，将喜欢的字体样式应用到签名中，让您的锁屏独具一格、个性十足。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1.  进入图库，选择图片，点击 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202504201ngpds/zh-cn/image/common/buttons/ic_more.png" width="24" height="24"/> > 设置为壁纸。
2.  点击时钟区域，选择艺术签名，智慧构图将根据壁纸类型，在部分壁纸自动推荐具有景深效果的锁屏艺术签名。
3.  点击艺术签名区域，进行个性化设置：
    +   点击艺术签名下方的文本框，输入签名文本，6 个中文以内效果最佳。
    +   点击<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202504201ngpds/zh-cn/image/common/buttons/textformat_size_square_fill-01.png" width="24" height="24"/>更换字体样式，还可点击<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202504201ngpds/zh-cn/image/common/buttons/ic_ai_font_12-01.png" width="24" height="24"/>选择带字的图片，AI 创作字体后一键应用到签名。
    +   点击<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202504201ngpds/zh-cn/image/common/buttons/ic_desktop_color-01.png" width="24" height="24"/>更换字体颜色。
4.  点击应用 > 确定，完成设置。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

建议选择主体突出、背景纯净的图片，以生成更好效果的艺术签名壁纸。
