# 元气心情主题，锁屏奇趣解压

<iframe src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202503171esvl6/zh-cn/image/video/10044753_f011_MoodMatch.mp4#toolbar=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="360" height="486"> </iframe>

设置元气心情主题，在锁屏界面体验与 3D 表情的趣味互动，点一点、按一按，好玩又解压。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1.  进入设置 > 桌面和个性化，点击更多主题，在**心情盒子**中选择**元气心情**。
    
2.  点击自定义进入编辑界面，可更换背景样式和画面风格。
    
3.  点击应用，体验元气心情主题，在锁屏界面点亮屏幕：
    
    +   点亮屏幕时，系统自动变换趣味动画。
        
    +   点击或长按表情，可触发趣味互动。
        
    +   充电时有专属充电动画，且在系统响铃模式下会伴随充电音效。
        
    +   与另一台使用元气心情主题的设备在锁屏界面碰一碰，两台设备主题情绪互换，还有一定几率触发隐藏动画。
        
<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

您可通过调节系统铃声音量，调节该主题的互动音效音量。
