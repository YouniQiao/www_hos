# 图片编辑跨设备无缝接续

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/figure/20005815_f001_AppContinuation.png" alt="图片编辑跨设备无缝接续" width="560" height="" />

通过接续，可在其他设备上无缝衔接当前设备上的任务。如：可将手机美图秀秀中的修图任务平滑切换到平板，大屏搭配手写笔进行更多创意编辑；处理完成后可切回到手机保存与分享。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/EMUI/C00B030/resource/card/202503041becsx/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1. 确保手机、平板登录同一**华为账号**，并均已安装**美图秀秀**。
2. 手机、平板进入**设置 > 多设备协同 > 接续**，开启**接续**开关。
3. 在手机上打开**美图秀秀**开始修图，平板快捷操作栏会**弹出美图秀秀图标**，点击弹出的美图秀秀图标，可将手机上的修图任务接续到平板。
4. 点击手机快捷操作栏中弹出的美图秀秀图标，可将修图任务返回手机。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/EMUI/C00B030/resource/card/202508300vZjQz/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

* 美图秀秀中的部分功能（如拼图、视频编辑）暂不支持接续，请以实际为准。
* 此功能目前仅支持在 HarmonyOS 5 及以上版本的设备间使用。
