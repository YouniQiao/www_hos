# AI辅助构图，推荐拍照角度

<iframe src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/2025033026fVij/zh-cn/image/video/20005877_f006_AI_composition.mp4#toolbar=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="360" height="486"> </iframe>

拍摄人物、建筑、自然风光等对象时，您可开启 AI 辅助构图，当手机识别到画面中有值得推荐的取景区域或角度时，将引导您调整拍摄，帮您发掘更多精彩拍摄视角。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1.  打开相机，选择拍照、人像或夜景模式 。
    
2.  在快门下方点击<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/2025033026fVij/zh-cn/image/common/buttons/ic_camera_up.png" width="24" height="24"/>，调出百宝箱，点击<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/2025033026fVij/zh-cn/image/common/buttons/ic_AI_composition.png" width="24" height="24"/>开启 AI 辅助构图。
    
3.  将镜头对准拍摄对象，根据拍摄对象操作：
    +   **人物**：如有推荐拍摄角度，取景框中会显示黄色和白色圆圈，根据提示调整手机的俯仰角度，直至两个圆重合。调整角度后，如有推荐取景区域，取景框中会显示中心环和推荐目标点。
    +   **风景**：如有推荐取景区域，取景框中会显示中心环和推荐目标点。
4.  移动手机，让中心环与目标点重合，直至目标点变为黄色。持稳手机并保持静止，此时相机会自动变焦，调整构图。
5.  待调整完毕后，点击快门拍照。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

+   如需拍摄风景和人像时辅助构图，建议您将相机焦段保持在 1x、2x 或 4x，或 1x、2x 或 3.7x （取决于您的机型）；如需拍摄人像时推荐角度，焦段保持在 1x。
+   如拍摄背景杂乱、或未识别到有更好的取景区域时， AI 辅助构图可能不会进行推荐，请以实际为准。
