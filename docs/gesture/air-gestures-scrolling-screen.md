# 隔空手势滑动屏幕

<video controls src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/video/vid_Swing_swipe.mp4" width="720" height="540"></video>

浏览内容时，若双手不方便直接操作，可用手势隔空上下滑动屏幕。

![方法](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

1. 单击桌面底部**快捷栏**中的 ![设置](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/icon/appicon_settings.png) 打开**设置**，单击左侧边栏的**系统**。
2. 在右侧界面，单击 **快捷启动和手势 > 隔空滑动屏幕**，确认**隔空滑动屏幕**的开关已开启。
3. 亮屏时，在屏幕前约半臂距离（建议 40~70cm 之间），将手掌朝向屏幕（指尖向上）或手背朝向屏幕（指尖向下）对准摄像头并稍作停顿，待屏幕出现 ![手势向上](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_AirGesture_handup_blue.png) 或 ![手势向下](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_AirGesture_handdown_blue.png) 后，向下或向上连续挥动手腕，即可让浏览页面向下或向上滑动。

![提示](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_tips.png) 提示

此功能要求电脑系统版本为 HarmonyOS 5.1 及以上。
