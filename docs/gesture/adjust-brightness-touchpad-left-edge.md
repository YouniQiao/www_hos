# 单指沿左边缘上下滑动以调节亮度

<video controls src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/video/vid_Touchpad_brightness_adjust.mp4" width="720" height="540"></video>

通过触控板手势，快速调节屏幕亮度。


![方法](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

单指沿着触控板的左边缘向上或向下匀速滑动，可增加或降低屏幕亮度。


![提示](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_tips.png) 提示

此功能要求电脑系统版本为 5.1.0.320 SP23 及以上版本。
