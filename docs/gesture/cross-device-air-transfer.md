# 跨设备隔空传送文件

<video controls src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/video/vid_Swing_GrabDrop.mp4" width="720" height="540"></video>

使用隔空手势，可轻松将本设备图库中的图片或视频、正在播放的音乐或视频、当前界面截图，隔空传送至其他设备上。

![方法](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

1. 在双方设备上，进入 **设置 > 系统 > 快捷启动和手势 > 隔空传送**，开启 **隔空传送** 的开关，并保持亮屏解锁。
2. 在发送设备上，打开要分享的内容：
   - **分享图片或视频：** 在 **图库** 中打开单个图片或视频。
   - **分享音乐：** 将华为 **音乐** 升级至最新版本后，打开歌曲的播放页面。
   - **分享视频：** 将华为 **视频** 升级至最新版本后，打开视频的播放页面。
3. 在发送设备屏幕前约半臂距离（建议 40~70cm 之间），伸开手掌对准摄像头并稍作停顿，待屏幕出现 ![手势图标](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_AirGesture_handup_blue.png) 后，握拳抓取待分享动效出现。
4. 保持握拳移动至接收设备屏幕前对准摄像头并稍作停顿，待屏幕出现 ![握拳图标](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_AirGesture_fist_blue.png) 后，伸开手掌开始传送。

![提示](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_tips.png) 提示

- 此功能要求电脑系统版本为 HarmonyOS 5.1 及以上。
- 仅部分手机/平板支持隔空传送，且系统版本需为 HarmonyOS 5 及以上。
- 隔空传送音乐、视频前，请在应用市场将华为 **音乐**、**视频** 升级至最新版本。
