# 单指沿右边缘上下滑动以调节音量

<video controls src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/video/vid_Touchpad_volume_adjust.mp4" width="720" height="540"></video>

通过触控板手势，快速调节系统音量。

![method](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

单指沿着触控板的右边缘向上或向下匀速滑动，可提高或降低系统音量。

![tips](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_tips.png) 提示

此功能要求电脑系统版本为 5.1.0.320 SP23 及以上版本。
