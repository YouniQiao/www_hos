# 潜水

> 本文适用于 **WATCH Ultimate 非凡大师**。

WATCH Ultimate 非凡大师支持百米深潜功能，满足专业潜水需求。

## 使用潜水功能

1. 在手表应用列表中，点击**潜水**。
2. 选择潜水模式（水肺潜水或自由潜水）。
3. 按照屏幕提示完成潜水前的安全检查。
4. 点击**开始潜水**，手表将实时记录潜水数据。

## 潜水数据

潜水过程中，手表会记录以下数据：

- 当前深度
- 水下时间
- 水温
- 上升速度
- 安全停留提醒

:::tip 提示
潜水后请查看手表上的潜水日志，了解详细的潜水数据分析。
:::

:::warning 注意
潜水是一项高风险运动，请确保您已接受专业培训并遵守潜水安全规范。手表仅作为辅助工具，不能替代专业潜水设备。
:::
