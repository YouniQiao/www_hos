# 北斗卫星通信

> 本文适用于 **WATCH Ultimate 非凡大师**。

海豚声纳通信（简称：海豚通信）。海豚通信功能开启后，可在潜水时通信使用。


海豚通信可实现潜伴之间通过手表在水下发送预置文字、表情消息、SOS求救消息 。海豚通信发送时，手表会在25秒内持续发出蜂鸣声，海豚最远通信距离可达30 米，水下SOS求救消息可通过开启海豚声呐通信功能的手表进行一次转发，最远通信距离达60米。




海豚通信以水为介质传递消息

- 在空气中只能短距离测试。
- 不支持空气和水之间发送接收消息。
