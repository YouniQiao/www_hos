# 潜水设置

> 本文适用于 **WATCH Ultimate 非凡大师**。

在开始潜水前，您可以根据个人需求调整潜水相关设置。

## 进入潜水设置

1. 在手表应用列表中，点击**潜水**。
2. 点击**设置**图标，进入潜水设置页面。

## 设置项说明

### 潜水模式

- **水肺潜水**：适用于使用气瓶的潜水方式。
- **自由潜水**：适用于闭气潜水方式。

### 气体混合

- 设置氧气浓度（21%~100%）。
- 设置氮气浓度。
- 设置氦气浓度（三混气潜水）。

### 安全提醒

- **深度提醒**：设置目标深度，到达时手表震动提醒。
- **上升速度提醒**：上升过快时手表发出警告。
- **安全停留**：到达指定深度后提醒进行安全停留。

### 其他设置

- **水下心率预警**：设置心率上下限。
- **水温显示单位**：摄氏度或华氏度。
- **深度显示单位**：米或英尺。

:::tip 提示
建议在潜水前仔细检查所有设置项，确保参数符合您的潜水计划。
:::
