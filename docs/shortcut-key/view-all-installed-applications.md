快捷键以查看所有安装应用

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

点按（键盘上）鸿蒙键![](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/keyboard/HM_keyboard_OH_1.png) + A 键，进入应用中心，可查看电脑中所有已安装的应用，单击应用图标即可打开。再次点按可退出。
