## 测量单次血氧饱和度
1. 正常佩戴穿戴设备，保持静止状态。
2. 在穿戴设备应用列表中，上下滑动屏幕，选择。
3. 点击，具体请以实际产品操作为准。
4. 测量过程中保持身体静止，屏幕朝上，即可测量当前的血氧饱和度。
## 自动血氧检测
开启自动检测后，静止状态下设备会自动测量和记录血氧饱和度，其数据可辅助观察健康状况（如处于睡眠时）。
1. 在的设备详情页，点击，。
2. 点击，可设置在非睡眠状态下血氧饱和度低于设置值时进行提醒。
## 查看数据
