# 记录女性生理周期

> 本文适用于 **WATCH Ultimate 非凡大师、WATCH GT 6 系列和WATCH 5**。

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="ultimate" label="WATCH Ultimate">


## 激活应用
首次使用时，请确保手表和手机连接正常，并完成激活。
1. 进入的首页，点击卡片，下载特性包后，设置您的经期数据，点击。如果首页不显示生理周期卡片，点击首页添加到首页。 > 💡 **提示：** ：如果首页不显示生理周期卡片，点击首页添加到首页。
2. 激活后，手表会弹出激活提醒（佩戴状态下直接弹出，非佩戴状态存入消息中心）。点击，进入应用首页。 点击，返回上一级。
## 设备侧记录和查看生理周期
1. 在穿戴设备应用列表中，选择，进入生理周期首页，点击侧更换图标 <img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000002520040368.png" width="24" />
，可以切换月视图和环形图。在月视图中切换月份可以查看前一个周期和未来三个周期的预测经期数据。
2. 点击编辑图标 <img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000002520360226.png" width="24" />
，可以记录和时间。向上滑动屏幕，可以查看经期相关的更多信息。
## 华为运动健康应用记录生理周期
1. 进入的首页，点击卡片，在页面选择目标日期，在下方勾选或，设置经期开始时间和结束时间，也可在中记录身体情况等。
2. 在页面，点击右上角 <img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000001701298789.png" width="24" />
，可进行如下设置： 点击，开启/关闭，及需要设置的各提醒项开关，还可设置和。
3. 点击，点击按钮开启后，并保持夜间佩戴手表，将跟踪您的体温、心率、呼吸率数据来改进预测结果，预测将更准确。功能仅部分设备支持，如果您的设备没有该设置项，则表示不支持该功能，请以实际显示为准。 > 💡 **提示：** 点击，开启 ，开启后并保持夜间佩戴手表，将跟踪您的体温、心率、呼吸率数据来改进预测结果，预测将更准确。
4. 点击，查看和等。
5. 点击，查看历史数据。
