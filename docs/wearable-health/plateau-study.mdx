# 高原健康研究

> 本文适用于 **WATCH Ultimate 非凡大师、WATCH GT 6 系列和WATCH 5**。

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="ultimate" label="WATCH Ultimate">



当您有高原出行计划时，高原健康研究应用可以根据测量结果和主观感受收集综合评估您的高原反应风险，并给出健康指导和建议。
> 💡 **提示：** 手机升级HarmonyOS 5 版本后该功能暂不支持，能从表侧发起测量 无法从手机侧发起测量，请您耐心等待功能适配，感谢理解！
## 开启高原健康研究项目
单击手表上键，在应用列表中，点击 > ，点击。
## 测量高原健康
- 手机侧测量 1. 打开应用，进入高原健康研究项目，点击。 2. 在设备侧，根据界面测量教程提示，完成测量。 3. 测量结束后，手机界面显示测量结果。
- 设备侧测量 1. 打开手机应用，进入高原健康研究项目。 2. 单击手表上键，在应用列表中，点击 > ，点击，首次测量时，需要点击根据测量教程完成问卷填写和生理数据采集。 3. 测量结束后设备界面显示测量结果，可点击 <img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000002358363533.png" width="24" /> 
查看结果说明。
## 查看数据
打开应用，进入高原健康研究项目，可查看测量数据和历史数据。
测量成功后可以在设备侧看到测量结果，上滑页面可点击查看统计详情和历史测量结果。

</TabItem>
