# 高原关爱

> 本文适用于 **WATCH Ultimate 非凡大师、WATCH GT 6 系列和WATCH 5**。

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="ultimate" label="WATCH Ultimate">


当处于高原地区时，高原关爱可帮助您实时了解自身身体状况，以便决定安排休息还是继续进行科学的运动，在很大程度上避免发生意外和高原反应带来的伤害。
> 💡 **提示：** 使用此功能前，请将您的设备升级至最新版本。 高原关爱适用于海拔2500米及以上的高原地区。 本产品非医疗器械，测量数据和结果仅供参考，不作为诊断和治疗的依据。 使用此功能时，请同时开启功能。
## 在手机侧开启高原关爱
在手机侧开启高原关爱后，当您进入高原地区，手表连续健康监测将启动，识别到高原适应性风险时会进行提醒。
进入的设备详情页，点击，开启。
## 在手表侧进行高反评估
进入高原地区后，当设备监测到疑似高原适应性风险、短时间内海拔变化过快或进入极高海拔地区（高于 4500 米）时，会提示是否要进行高反评估，您可选择进行高反评估或忽略提醒。
> 💡 **提示：** 疑似高原适应性风险提醒和海拔变化过快提醒仅支持在海拔2500-4500米的范围。
您还可手动启动高反评估：
1. 单击手表上键，在应用列表中点击。
2. 点击，根据提示完成问卷自检、血氧测量。
3. 评估完成后，在高原关爱首页可查看评估结果。
