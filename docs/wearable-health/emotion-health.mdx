# 情绪健康

> 本文适用于 **WATCH Ultimate 非凡大师、WATCH GT 6 系列和WATCH 5**。

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="ultimate" label="WATCH Ultimate">



> 💡 **提示：** 测量数据和结果仅供参考，不作为诊疗依据，如身体不适建议您及时就医。
## 开启情绪/压力辅助记录
1. 在设备应用列表中，点击。根据界面提示完成相关操作后，进入页面。
2. 在的设备详情页，点击，打开开关。
## 查看/修改情绪数据
在设备侧查看
在设备应用列表中，点击，通过情绪动效图显示此时的情绪状态，向上滑动可查看更多记录数据。
在华为运动健康查看
进入首页，点击，即可查看您的情绪/压力数据，同时提供不同时间段的分析和解读建议。
查看历史所有数据，进入首页，点击，点击上角 > ，即可查看。
## 情绪提醒
1. 在设备应用列表中，点击。
2. 向上滑动屏幕至页面，点击，打开开关，设备会根据您的最近情绪状态进行消息提醒。当最近一段时间持续情绪不佳时，可根据提醒内容中的建议，帮助您调整情绪状态。
