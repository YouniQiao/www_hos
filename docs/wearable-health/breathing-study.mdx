# 呼吸健康研究

> 本文适用于 **WATCH Ultimate 非凡大师、WATCH GT 6 系列和WATCH 5**。

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="ultimate" label="WATCH Ultimate">



> 💡 **提示：** iOS手机无华为创新研究应用，与 iOS手机配对使用时不支持在手机侧使用，可在手表端独立使用。 测量数据和结果仅供参考，不作为诊疗依据，如身体不适建议您及时就医。
## 开启呼吸健康研究项目
1. 在中配对连接设备后，打开手机侧。
2. 点击设备侧，首次进入时，需同意隐私声明。
3. 打开 > ，点击。
## 单次测量呼吸健康
- 手机侧测量 1. 打开手机侧华为 > ，点击。 2. 在设备侧，根据界面测量教程提示，完成测量。
- 设备侧测量 1. 打开手机侧，进入呼吸健康研究项目。 2. 单击设备的上键，在应用列表中点击 > > ，首次测量时，需点击根据测量教程完成咳嗽音录制和生理数据采集。 3. 测量结束后设备界面显示测量结果，可点击 <img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000002344579578.png" width="24" /> 
查看结果说明。
## 夜间睡眠期间测量
1. 打开手机侧 > ，点击右上角菜单，点击，打开。
2. 开启后，在您睡眠期间会评估您的呼吸健康情况，在设备连接到手机后，手机侧下拉页面可将您的呼吸健康状况同步到手机中。
## 查看数据
打开 > ，可查看测量数据和历史数据。
测量成功，您可以到手机侧下拉同步测量数据。同步后可以看到测量结果和呼吸率，上滑页面可点击查看统计详情和历史测量结果。
