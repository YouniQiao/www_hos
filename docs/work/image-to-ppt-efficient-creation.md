# 图片转PPT，助您高效创作

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/figure/fig_AI_ImageTransfer_PPT.png" alt="图片转 PPT" width="720" height="540" />

将喜爱的 PPT 图片发送给小艺，让小艺帮您转换成可编辑的 PPT 文档，助您高效创作。

![方法](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

1. 单击桌面右下角 ![小艺](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/icon/appicon_xiaoyi.png)，打开小艺侧边窗。
2. 在应用界面（如文件管理、图库等）选择 PPT 内容样式的图片，单指长按触控板（或鼠标左键）并移动，将其拖拽至小艺侧边窗区域，然后松开手指。
3. 单击 Tips 提示：**图片转ppt**。若未显示 Tips 提示，可以发送文本或语音给**小艺**，如“转 PPT”。
4. 转换完成后，可选中 PPT 文档右键选择**复制**或**保存**等方式，将生成的文档保存至本机。

![提示](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_tips.png) 提示

并非图片中所有内容都可转换为可编辑的内容格式，请以实际情况为准。
