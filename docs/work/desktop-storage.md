# 收纳夹，桌面灵活布局

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/figure/fig_desktop_zone.png" alt="收纳夹，桌面灵活布局" width="720" height="540" />

使用**收纳夹**，桌面一屏可容纳更多内容，您还可以调节收纳夹大小，灵活布局您的桌面；单击收纳夹中的图标可快速打开应用、文件夹、文件等。

![方法](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

1. 在电脑桌面空白处，右键选择**新建 > 收纳夹**，单击收纳夹下方的默认名称文本进入编辑模式，可修改收纳夹名称。
2. 长按待移入收纳夹的图标，将其拖拽至收纳夹区域后松开手指。
3. 您还可以根据需要，体验以下操作：
   - **调整收纳夹位置：** 光标移至收纳夹空白处，单指长按触控板（或鼠标左键）并移动，将其拖拽至桌面任意位置（不支持与桌面卡片、其他收纳夹的位置重叠，若拖拽后回弹为正常现象）。
   - **调整收纳夹尺寸：** 光标移至收纳夹边缘，当光标变化成双向箭头时（如 ![上下箭头](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_cursor_NorthSouth.png)、![对角箭头](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_cursor_NorthWest_SouthEast.png)），单指长按触控板（或鼠标左键）并移动，可拖拽边缘调整尺寸。
   - **调整收纳夹内图标位置：** 光标移至收纳夹中某图标区域，单指长按触控板（或鼠标左键）并在收纳夹内移动，出现矩形框时松开手指。
   - **收纳夹内图标自动排序：** 光标移至收纳夹空白处，右键选择**排序**，根据需要选择想要的排序规则，如名称、修改日期、创建日期等。
   - **快速直达：** 双击收纳夹中的图标，可直接打开应用、文件夹、文件等。

![提示](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_tips.png) 提示

桌面卡片不支持移至收纳夹，请以实际情况为准。
