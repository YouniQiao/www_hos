# 图片编辑跨设备无缝接续

<iframe src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202504201ngpds/zh-cn/image/video/20005815_f001_AppContinuation.mp4#toolbar=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="360" height="486"> </iframe>

通过接续，可在其他设备上无缝衔接当前设备上的任务。如：可将手机美图秀秀中的修图任务平滑切换到平板，大屏搭配手写笔进行更多创意编辑；处理完成后可切回到手机保存与分享。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1.  确保手机、平板登录同一华为账号，并均已安装美图秀秀。
2.  手机、平板进入设置 > 多设备协同 > 接续，开启接续开关。
3.  在手机上打开美图秀秀开始修图，平板快捷操作栏会弹出美图秀秀图标，点击弹出的美图秀秀图标，可将手机上的修图任务接续到平板。
4.  点击手机快捷操作栏中弹出的美图秀秀图标，可将修图任务返回手机。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

+   美图秀秀中的部分功能（如拼图、视频编辑）暂不支持接续，请以实际为准。
+   此功能目前仅支持在 HarmonyOS 5 及以上版本的设备间使用。
