# 用机有疑问？试试找小艺

<iframe src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/2025102603wmgm/zh-cn/image/video/10044717_f011_celiacontrol.mp4#toolbar=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="360" height="486"> </iframe>

使用手机的过程中，当您对功能操作有疑问时，让小艺为您解答，还可通过小艺快速执行操作。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

长按屏幕底部导航条，唤醒小艺，说出对功能操作的疑问，如：“怎么把华为分享设置为所有人可发现”。

+   点击答案中的蓝色字体，可以直达设置项。
    
+   点击答案下方的小艺帮帮忙，可让小艺帮您执行操作。
    

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

使用三键导航时，可通过长按电源键、语音唤醒等方式唤醒小艺。
