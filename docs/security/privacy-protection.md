# 防窥保护，守护隐私安全

<iframe src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202504201ngpds/zh-cn/image/video/20012156_f001_ScreenPrivacy.mp4#toolbar=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="360" height="486"> </iframe>

在公共场合查看手机中的隐私信息时，您可开启防窥保护，若有被他人窥视的风险，手机会提醒您留意周围环境，部分应用还可设置自动隐藏界面内容。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

进入设置 > 隐私和安全 > 防窥保护，开启防窥保护开关，通过人脸验证后，打开需要加入保护的应用开关。开启后，手机支持以下保护：

+   **查看加入保护的应用**：有被窥视风险时，手机将通过实况窗消息提示，部分三方应用还可设置自动隐藏界面。
+   **查看图库隐藏相册**：有被窥视风险时，手机将隐藏相册界面。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

+   本功能通过低功耗传感器智能判断您的周边环境，仅当机主本人浏览指定应用、并存在被他人窥视的风险时，系统才会自动提示。识别过程均在设备端完成，不会泄露您的隐私。
+   暗光、强光、逆光或角度过大等场景可能会影响该功能的识别成功率。
