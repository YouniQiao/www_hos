# 安全访问，保护隐私数据

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/2025033026fVij/zh-cn/image/figure/20005744_f002_PrivacyPicker.png" width="360" height="486"/>

您的手机默认支持安全访问功能，当应用需要访问图片、视频、联系人等敏感数据时，安全访问可控制应用仅访问您选定的数据，保护您的数据安全。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

以在应用中发送或上传图片为例，在安全访问的限制下，应用仅可访问您选定的内容，无法访问其余未选定的数据，也无法在后台持续访问图库。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

在应用访问数据后，您可在设置 > 隐私和安全中，点击图片和视频等选项，查看应用的访问记录。
