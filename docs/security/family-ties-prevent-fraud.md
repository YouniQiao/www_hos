# 亲情防诈，为家人守护设备安全

<iframe src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202504201ngpds/zh-cn/image/video/20012126_f002_SmartProtection.mp4#toolbar=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="360" height="486"> </iframe>

开启亲情防诈，当家人疑似遇到诈骗风险时，手机会及时向您发送通知，您可及时进行干预，守护家人用机安全。


<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1.  在您的手机和家人设备上，登录各自的华为账号，并在设置 > 健康使用设备中绑定守护。
2.  绑定后，亲情防诈将自动开启，守护以下场景：
    +   **接打风险电话**：您将同步收到提醒，并可为家人挂断通话。
    +   **收到风险短信、访问风险网页**：您将收到风险记录。
    +   **安装风险应用**：您将同步收到提醒。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

在您的手机上进入设置 > 隐私和安全 > 骚扰和诈骗拦截 > 亲情防诈，可查看亲情防诈记录。
