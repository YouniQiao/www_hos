# 智能隐藏横幅通知内容

<iframe src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/2025102603wmgm/zh-cn/image/video/10044804_f009_SmartReminder.mp4#toolbar=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="360" height="486"> </iframe>

解锁屏幕后，当系统检测到非机主或多人查看手机时，会自动隐藏横幅通知内容，保护机主的隐私安全，若佩戴华为手表，则由手表提醒通知。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1.  进入设置 > 生物识别和密码 > 人脸识别，根据界面提示录入人脸信息，开启人脸解锁。
    
2.  进入设置 > 通知和状态栏 > 横幅通知，在显示横幅通知内容中，选择仅机主注视时选项。
    

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

+   使用该功能前，请将您的设备升级到最新版本。
    
+   由手表提醒通知需满足：佩戴的手表与手机已蓝牙配对、且在手机运动健康应用已开启同步手机消息通知至手表开关和单设备提醒模式。
    
+   仅部分手表支持该功能，请以实际为准。
    
+   针对地震、通话等紧急消息，手机和手表均会收到提醒。
