# 一键星盾防诈，守护用机安全

<iframe src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202612210iZdjk/zh-cn/image/video/20012126_f003_SmartScamCheck.mp4#toolbar=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="360" height="486"> </iframe>

开启星盾防诈，智能识别通信、网页、应用等多场景诈骗风险，多方位守护用机安全。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202612210iZdjk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

进入设置 > 隐私和安全 > 星盾防诈，根据提示，一键开启星盾防诈，以下防护自动开启：

+   **通信防护**：增强通话和短信检测，如视频通话时，检测对方是否使用 AI 换脸等。

+   **诈骗工具防护**：增强网页、应用和 WLAN 检测，如禁止诈骗应用安装等。

+   **涉诈场景防护**：增强日常风险场景检测，手机会自动检测并识别涉诈场景，及时提醒并拦截。

在星盾防诈界面，点击防诈提醒记录图，可查看近 30 天的防护记录。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202612210iZdjk/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

+   使用该功能前，请将您的设备升级到最新版本。
