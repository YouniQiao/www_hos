# 设备丢失，星闪精确查找

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/2025033026fVij/zh-cn/image/figure/10044536_f001_FindPhone.png" width="360" height="486"/>

开启查找设备功能后，通过星闪查找，实现精准测距和方位指向，让您快速找回丢失的华为设备。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

以查找端和被查找端均为华为手机举例：

1.  查找端开启星闪 | 蓝牙开关，打开查找设备应用，点击需查找的设备进入详情页。
2.  点击精确查找，当进入星闪查找范围（最大 50 米范围）时，根据信号强弱，将显示距离和方位。
3.  根据距离和方位指引，如已连接成功可尝试播放铃声提醒，直至找到丢失的设备。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

查找和被查找的手机均需满足：登录相同华为账号、开启查找我的手机并完成安全认证。
