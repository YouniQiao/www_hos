# 更换手表或更换eSIM号码

> 本文适用于 **WATCH Ultimate 非凡大师和WATCH 5**。



如需更换手表或更改eSIM号码，需先注销已开通的eSIM业务，并在手表eSIM管理界面内执行重置操作，然后重新开通。
1. 联系运营商或按以下方式注销已开通的eSIM号码，否则该号码无法再次开通，且会持续扣费。
2. 使用新手机号或在新手表上开通eSIM业务。
> 💡 **提示：** 重置操作仅删除手表中的eSIM信息，不会自动注销运营商eSIM业务。运营商eSIM业务未完成注销会持续收费。注销方法请联系运营商热线咨询： 中国移动：10086 中国联通：10010 中国电信：10000
