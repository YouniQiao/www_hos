# 中国联通用户注销独立号码

> 本文适用于 **WATCH Ultimate 非凡大师和WATCH 5**。

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="ultimate" label="WATCH Ultimate">



> 💡 **提示：** 手机升级HarmonyOS 5 及以上版本后该功能暂不支持，请您耐心等待功能适配，感谢理解！ 广东广州暂不支持开通，详情请咨询10010。
注销独立号码业务需要先前往中国联通营业厅注销业务，再在手表端删除eSIM配置文件。
1. 前往中国联通营业厅注销eSIM独立号码业务。
2. 点击手表上键，在应用列表中选择 > > > 。
3. 在设置页面点击，根据界面提示完成重置，删除eSIM相关配置文件。
