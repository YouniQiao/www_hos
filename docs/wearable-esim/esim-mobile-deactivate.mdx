# 中国移动用户注销一号双终端业务

> 本文适用于 **WATCH Ultimate 非凡大师和WATCH 5**。



开通eSIM一号双终端业务后，您可通过App管理您的一号双终端业务，注销一号双终端业务需要在App中完成注销，并在手表端删除eSIM配置文件。华为手机还可以通过注销一号双终端业务。
## 通过App注销一号双终端业务：
1. 使用主号码登录App。
2. 在App中点击 > ，根据界面提示完成操作，即可注销已开通的一号双终端业务。
3. 等待注销完成，注销成功后，会收到短信提醒。
## 在手表端删除eSIM配置文件：
1. 点击手表上键，在应用列表中选择 > > > 。
2. 在设置页面点击，根据界面提示完成重置，删除eSIM相关配置文件。
## 通过注销一号双终端业务
> 💡 **提示：** 仅华为手机支持通过华为运动健康注销一号双终端业务。
1. 打开，进入设备详情页，点击 > 。
2. 在已开通界面点击，点击，根据界面提示完成操作，即可注销已开通的一号双终端业务。
3. 等待注销完成，注销成功后，会收到短信提醒。
