# 中国联通用户开通独立号码

> 本文适用于 **WATCH Ultimate 非凡大师和WATCH 5**。



> 💡 **提示：** 近期各运营商均在进行业务升级，运营商最新支持策略请查看：HUAWEI WATCH 系列 eSIM开通方式。 需要有联通实体SIM卡，没有会影响为手表开通独立号码业务。
eSIM独立号码是为手表开通全新的号码和蜂窝移动网络，拥有单独的资费套餐，拥有短信、通话和网络连接等业务。开通后，手表与手机蓝牙连接时，可选择用手机卡或eSIM卡拨号；手表与手机蓝牙断连时，则通过手表eSIM卡拨号。
运动健康App在线开通：
打开，进入设备详情页，点击 > > > ，根据界面提示进行开通。
中国联通App手机营业厅开通：
HarmonyOS 4.3及以下版本手机/安卓手机：从中国联通App手机营业厅开通，具体操作请按照联通App的界面指引操作完成。
HarmonyOS 5及以上版本手机/iOS手机：请先使用HarmonyOS 4.3及以下版本手机/安卓手机从中国联通App手机营业厅开通，然后再连接原手机使用，注意需使用同一华为账号登录。
中国联通线下营业厅开通：
中国联通独立号码业务需前往中国联通营业厅办理，请您前往营业厅后，参考如下步骤或根据现场工作人员指引完成开通。
1. 在的设备详情页，点击 > > > ，选择线下营业厅开通。
2. 根据界面提示，完成如下操作： 请携带本人有效身份证与手表，前往运营商营业厅办理eSIM独立号码业务。
3. 点击，使用手机扫描运营商提供的二维码，下载eSIM信息。
