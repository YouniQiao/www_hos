# 查看设备信息

> 本文适用于 **WATCH Ultimate 非凡大师和WATCH 5**。

## 在手表侧查看

1. 单击手表上键，在应用列表中点击**设置**，进入设置页面。
2. 点击**设备的名称（如，HUAWEI WATCH XXX）**，在**关于手表**页面上滑屏幕。
可查看手表的**型号名称**、**型号代码**、**软件版本**、**序列号**、**蓝牙地址**、**IMEI** 等信息。



手表EID 号查询（仅全能模式下可查询）：单击手表上键，在设备应用列表中点击**设置 > 移动网络** > **eSIM 管理** > **设置** ，在设置界面可查看EID号。



## 在手机侧查看


在**华为运动健康**的设备详情页，点击**通用设置 > 设备信息**，可查看**设备名称**、**设备型号**、**设备 SN 号**、**设备 MAC**即**蓝牙地址**。
