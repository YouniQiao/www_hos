# 多任务管理

> 本文适用于 **WATCH Ultimate 非凡大师、WATCH GT 6 系列和WATCH 5**。

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="ultimate" label="WATCH Ultimate">



实现自由切换最近应用，快速清除所有后台任务。
1. 在手表亮屏解锁状态下，双击上键，进入多任务管理界面，可以查看当前打开的所有应用。
2. 您可以左右滑动屏幕，进行下列操作： 点击任意应用卡片，进入应用详情页面。
3. 按住任意应用卡片上滑，删除任务。
4. 点击 <img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000001628586178.png" width="24" /> 
，删除所有任务。
> 💡 **提示：** 运行中的锻炼任务，不支持删除。

</TabItem>
