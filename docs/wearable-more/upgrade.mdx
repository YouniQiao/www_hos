# 升级版本

> 本文适用于 **WATCH Ultimate 非凡大师、WATCH GT 6 系列和WATCH 5**。

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="ultimate" label="WATCH Ultimate">



## 在设备端升级
> 💡 **提示：** 与iOS手机配对连接时，不支持在设备端升级设备版本。 穿戴产品系统版本升级提示不支持关闭，不支持版本回退。
1. 在设备应用列表中，点击 > > 。
2. 点击更新图标，根据提示进行下载更新操作。
## 在手机端升级
1. 在的设备详情页，点击 > 。
2. 在页面，点击上角图标，打开和开关，有新版本时会静默传输至设备侧进行版本更新。iOS手机路径为：在页面，点击上角图标，打开和开关。
3. 手机将检测设备的版本更新信息，请根据提示进行更新操作。

</TabItem>
