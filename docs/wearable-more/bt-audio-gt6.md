# 连接蓝牙耳机/音箱

> 本文适用于 **WATCH GT 6 系列**。

手表与蓝牙耳机/音箱正常连接后，可通过已连接设备听音乐等。
## 通过手表“设置”连接蓝牙耳机/音箱
1. 手表与配对连接后，将蓝牙耳机/音箱设置为可配对状态。
2. 进入手表应用列表，上下滑动屏幕，选择 > 。
3. 选择目标蓝牙耳机/音箱连接。
## 通过手表“音乐”连接蓝牙耳机/音箱
1. 手表与配对连接后，将蓝牙耳机/音箱设置为可配对状态。
2. 进入手表应用列表，上下滑动屏幕，选择 ，在音乐播放界面点击右下角 <img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0ER/zh-CN/zh-cn_image_0000002432867644.png" width="24" /> ，点击 > 。

3. 选择目标蓝牙耳机/音箱连接。
