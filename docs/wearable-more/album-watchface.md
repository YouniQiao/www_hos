# 设置相册表盘

> 本文适用于 **WATCH 5**。

## 设置相册表盘

1. 进入**华为运动健康**的设备详情页，点击**我的表盘 > 相册表盘**，进入相册表盘设置界面。
2. 点击<img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EB/zh-CN/zh-cn_image_0000002356284902.png" width="24" />，选择**拍照**或**选择图片**，选择照片后，拖动照片调整显示的区域，点击右上角的<img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EB/zh-CN/zh-cn_image_0000002356437526.png" width="24" />，照片上传完成。
3. 在**相册表盘**页面，可设置表盘的**颜色**、**样式**、**功能**等。
4. 在**其他设置**中，开启或关闭**抬腕显示新照片**。
当上传图片大于一张时，开启后每次抬腕亮屏会切换一张新照片，关闭后抬腕将停留在上次显示的照片，需要点击设备表盘的屏幕才会切换新照片。
5. 点击**设为当前表盘**，设备表盘将设置为相册表盘。


> 💡 **提示：** 相册表盘支持以下格式图片：*.gif、*.jpg、*.jpeg、*.png、*.bmp、*.psd、*.tif、*.tiff、*.webp、*.pcx，其中gif图显示为静图，不显示动态效果。
