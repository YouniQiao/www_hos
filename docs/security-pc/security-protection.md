# 安全防护，实时防护病毒威胁

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/figure/fig_settings_VirusProtection.png" alt="安全防护" width="720" height="540" />

使用**安全防护**，扫描设备内的病毒威胁，实时风险防护、保护设备安全。

![方法](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

1. 进入**设置 > 隐私和安全 > 高级 > 安全防护**，确认**安全防护**开关已开启（默认开启）。开启后，将实时防护设备风险，发现病毒威胁时将以通知方式进行提醒。
2. 您也可以单击桌面底部**快捷栏**中的 ![文件管理](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/icon/appicon_filemanager.png) 打开**文件管理**，选中待检测的文件夹或文件，右键选择**更多 > 使用安全防护检测**。
3. 检测完成后，若安全防护界面有风险文件提示，可在**未处理的风险**列表中，单击风险文件右侧的**删除**或**忽略**。
