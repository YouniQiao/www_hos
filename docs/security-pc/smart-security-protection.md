# 智能安全防护，守护通话安全

<iframe src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202602261wvwjj/zh-cn/image/video/vid_FaceSwap_Check.mp4#toolbar=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="720" height="540"> </iframe>

智能安全防护，可在视频通话时智能检测换脸痕迹，当检测到对方可能存在 AI 换脸风险时会弹出提示，帮您防范潜在的诈骗风险。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202602261wvwjj/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1. 单击桌面底部快捷栏中的设置图标打开设置，单击左侧边栏的隐私和安全。

2. 在右侧界面，单击骚扰和诈骗拦截 > 换脸检测，并单击开启视频通话时检测的开关。首次开启，请根据界面提示联网下载 AI 换脸检测插件。

3. 开启后，进行视频通话时将自动检测人脸信息，当检测到对方可能存在 AI 换脸风险时，通过弹框提示风险。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202602261wvwjj/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

+ 部分应用可能不支持视频通话检测，请以实际情况为准。

+ 人脸检测过程仅在设备本地完成，不会将个人信息上传至云端。
