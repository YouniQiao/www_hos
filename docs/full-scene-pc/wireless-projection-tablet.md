# 无线投屏，协同平板扩展屏幕

<iframe src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202602261wvwjj/zh-cn/image/video/vid_wireless_projection_pad.mp4#toolbar=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="720" height="540"> </iframe>

无线投屏连接平板，扩展电脑的操作方式和空间，双屏操作、浏览更便捷。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202602261wvwjj/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1. 在电脑、平板上，登录同一华为账号。

2. 在电脑上，单击桌面右下角状态栏中间区域（包含电池图标）或点按（键盘上）鸿蒙键 + C 组合键，打开控制中心。

3. 单击投屏图标启动并搜索可投屏的设备。您也可以点按（键盘上）F8 键（Fn 键指示灯熄灭时），开启无线投屏。

4. 搜索完成后，在设备列表中单击待连接的平板名称，将电脑屏幕内容投送至平板。

5. 根据需要，选择投屏显示模式：

   - **扩展显示器：** 将平板作为电脑的扩展屏幕，可在电脑和平板打开不同的应用窗口，双屏操作。

   - **本机屏幕的镜像：** 平板与电脑屏幕显示相同内容并实时同步。

6. 投屏完成后，通过以下任一方式可退出投屏。

   - **电脑：** 单击桌面右下角 > 投屏图标，然后单击正在投屏中的设备名称退出投屏。

   - **平板：** 从屏幕顶部下滑，打开状态栏，点击投屏图标，然后点击退出投屏。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202602261wvwjj/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

+ 此功能要求双方设备升级至 HarmonyOS 6.1 及以上版本。
