# 多屏协同，打开多个手机窗口

<iframe src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202602261wvwjj/zh-cn/image/video/vid_pc_connect_phone_MultiWindow.mp4#toolbar=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="720" height="540"> </iframe>

电脑和手机协同连接后，可在电脑上打开多个手机窗口，一屏多用，处理任务更高效。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202602261wvwjj/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1. 在手机、电脑上，开启 WLAN、蓝牙，并登录同一华为账号。

2. 在电脑上，单击桌面右下角状态栏中间区域（包含电池图标）或点按（键盘上）鸿蒙键 + C 组合键，打开控制中心。

3. 在超级终端中单击，然后在设备列表中单击待连接的手机，发起协同连接。

4. 协同连接后，在电脑上的手机主窗口中打开一个应用，单击顶部的加号，即可打开该应用的分身子窗口。您可在手机主窗口继续打开其他应用，各窗口独立使用，互不影响。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202602261wvwjj/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

+ 使用该功能前，请将您的设备升级到最新版本。

+ 若打开应用的分身按钮置灰，说明该应用不支持分身或部分场景不支持分身（如设置了应用锁的应用），请以实际情况为准。

+ 分身子窗口有数量限制，达到上限时会提示"新窗口已达上限"。
