# 手眼同行，文件随目光拖放

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/figure/fig_KeyboardMouseSharing_DragFiles.png" alt="手眼同行，文件随目光拖放" width="720" height="540" />

键鼠共享时，拖动当前设备上的文档或图片，然后将视线转移至目标设备，按下 Ctrl 键确认，即可将文档或图片瞬间移动至目标设备。目光所及，素材即达，简单高效。

![方法](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

以从手机拖动图片瞬移至电脑备忘录为例：

1. 在手机、电脑上，开启 **WLAN**、**蓝牙**，并登录同一**华为账号**。
2. 在手机、电脑上，进入**设置 > 多设备协同 > 键鼠共享**，确认已完成以下设置：
   - **手机：**确认**键鼠共享**开关已开启（默认开启），并开启**手眼同行**的开关。
   - **电脑：**确认**键鼠共享**、**允许鼠标移动至附近设备屏幕以内连接设备**开关已开启（默认开启），并开启**手眼同行**的开关。
3. 在电脑上，打开**备忘录**并单击某备忘进入编辑模式，然后移动光标至屏幕边缘并持续向外移动或者光标连续轻触屏幕边缘两次，当屏幕边缘出现穿越动效时，持续向屏幕外移动光标，直至手机出现键鼠共享提示。
4. 鼠标光标穿越至手机后，单指长按触控板或鼠标左键在手机上拖动图片（保持长按不松手），然后移动视线至电脑屏幕，当上方出现 ![手眼同行图标](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_public_face.png) 时，点按电脑键盘上的 Ctrl 键确认。
5. 图片随光标穿越至电脑后，移动光标至备忘待插入图片的位置，当图片右上角出现 ![添加图标](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_public_add_norm_green.png) 时松开手指。
