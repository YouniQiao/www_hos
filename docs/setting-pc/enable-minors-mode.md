# 开启未成年人模式

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/figure/fig_settings_minors_mode.png" alt="开启未成年人模式" width="720" height="540" />

在设备上开启**未成年人模式**，部分应用将自动切换为**未成年人模式**，引导孩子浏览合适的内容，还可为孩子设置设备使用时间。

![方法](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

使用此功能前，请先登录**华为账号**。

1. 单击桌面底部**快捷栏**中的 ![设置](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/icon/appicon_settings.png) 打开**设置**。您也可以单击桌面底部**快捷栏**中的 ![应用中心](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/icon/appicon_allapps.png) 进入**应用中心**，单击打开**设置**。
2. 单击左侧边栏的**健康使用设备**，然后在右侧界面勾选**未成年人使用**并单击**开启**。
3. 您可选择已有使用者；或者单击**创建新的使用者**，设置孩子信息（头像、名称、年龄）后单击**同意并创建**。
4. 根据界面提示，设置未成年人模式的密码和密保问题。
5. 在未成年人模式页面，可为孩子设置设备和应用的使用时间，如屏幕使用的总时间、限制应用和元服务的使用时间、始终可用的应用和元服务等。
6. 开启该功能后，当打开部分视频类应用时，将默认进入**未成年人模式**，确保孩子访问内容安全。
