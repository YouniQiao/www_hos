# 小艺深度解题，金牌智能助教

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/figure/10104441_f003_celiasolve.png" alt="小艺深度解题" width="560" height="" />

遇到难题不会做，同一道题反复错？让小艺深度解题为您答疑，通过引导式解题启发自主思考，学习更高效。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/EMUI/C00B030/resource/card/202503041becsx/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1. 在桌面打开**小艺**应用，点击**广场** > <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_find.png" width="24" height="24"/> ，搜索“小艺深度解题”并进入。
2. 根据需要，选择合适的解题方式，如**可视化解题**。
3. 点击**可视化解题**，拍照识别题目后，智能体将生成图文并茂的解题视频。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/EMUI/C00B030/resource/card/202508300vZjQz/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

小艺深度解题还支持 **AI 互动讲解**、**AI 辅导写作**等不同解题方式。
