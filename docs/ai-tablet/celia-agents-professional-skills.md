# 小艺智能体，一步开启专业技能

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/figure/10104441_f002_celiasamrt.png" alt="小艺智能体" width="560" height="" />

小艺智能体可为您提供如翻译、深度解题等专业服务，您可根据需求选择不同智能体，还可将智能体添加至桌面，以便快速打开。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/EMUI/C00B030/resource/card/202503041becsx/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

以**小艺翻译助手**智能体为例：

1. 在桌面打开**小艺**应用，点击**广场** > <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_find.png" width="24" height="24"/>，搜索“小艺翻译助手”并进入。
2. 在“小艺翻译助手”中，您可按提示翻译所需内容。
3. 点击右上角 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/phone_ic_more.png" width="24" height="24"/> > **添加至桌面**，可创建**小艺翻译助手**桌面快捷方式。
