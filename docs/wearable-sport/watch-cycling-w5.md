# 手表骑行联动

> 本文适用于 **WATCH 5**。

专业运动


|  | 手表骑行联动 |
| --- | --- |


   手表骑行联动

    手表上开启骑行，可联动手机实时显示骑行数据，数据呈现更丰富。


> 💡 **提示：**
>
> - 开始运动前需先开启**手机查看数据**功能：在手表应用列表中选择**锻炼 > 锻炼设置** > **手机查看数据**，开启该功能。
>
> - 与HarmonyOS 5及以上版本的手机配对使用时，仅支持户外骑行在手机侧联动，室内单车暂不支持。
>
> - 为了保证在运动过程中手机显示骑行数据，蓝牙连接不能断开。
>
> - 手表配对三方踏频计手机上才会显示踏频数据。


1. 单击手表上键，在应用列表中点击**锻炼**。
2. 上下滑动屏幕选择点击**户外骑行**或**室内单车**。点击图标开始运动，手机侧同步显示骑行数据。在手机侧点击显示的数据类型，可根据需要选择切换。
