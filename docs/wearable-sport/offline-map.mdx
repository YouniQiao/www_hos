# 使用离线地图

> 本文适用于 **WATCH Ultimate 非凡大师、WATCH GT 6 系列和WATCH 5**。

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="ultimate" label="WATCH Ultimate">



在下载离线地图后，可在手表侧导航线路或户外运动中直接开启地图导航。离线地图可用于锻炼（户外运动、轨迹导入导出、轨迹返航等）等应用场景地图显示，手表开启户外运动后，需要定位成功才可以使用离线地图。
## 下载离线地图
1. 在的设备详情页，选择，进入离线地图界面。
2. 进入页面，选择所需地区的地图，点击。下载过程中，请勿离开本页面，下载完成后，可在页面查看。
**自动下载：**
> 💡 **提示：** 仅支持与HarmonyOS 4.3及以下、HarmonyOS 6 （系统版本为 6.0.0.112 及以上）版本的华为手机和 Android 手机配对时使用。
1. 在的设备详情页，选择，进入离线地图界面。
2. 在页面，开启开关。手机路径：在页面，开启开关。
3. 开启后，当手表检测到您到达新城市/地区，且手机已连接 WLAN 时，将自动下载所在位置的离线地图并同步至手表。
## 删除离线地图
1. 在的设备详情页，选择，进入离线地图界面。
2. 进入页面，选择需要删除的地图，点击右侧的 > ，可删除该离线地图。
## 地图设置
1. 单击手表上键，在应用列表中选择 > 。
2. 根据需要设置地图的显示信息。 ：、或。
3. ：或。
## 更多信息
使用离线地图在户外类运动的运动轨迹中增加了地图显示，在地图图层之上绘制运动轨迹；在运动中地图上绘制导入的线路轨迹、标记点、轨迹行进方向等指示；在线路导入导航后可在地图上预览导入的轨迹路线。
导航线路：
在设备应用列表中，点击，上下滑动屏幕选择某种户外运动，如，在运动准备页面，上滑屏幕至页面，选择。
