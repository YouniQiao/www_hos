# 开始锻炼

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="ultimate" label="WATCH Ultimate">


华为手表支持多种运动模式，包含跑步、骑行等专业运动模式和健身类、器械类、休闲类等其他运动模式，您可根据实际需要选择并开启锻炼。


## 在手表上启动

1. 在穿戴设备应用列表中，点击**锻炼**。
2. 上下滑动屏幕选择锻炼类型。也可以向上滑动屏幕至最后一页，点击**添加运动**，添加其他运动类型。
3. 点击锻炼类型图标，即可开始锻炼。
4. 运动状态下，长按上键可结束锻炼；或点击上键暂停锻炼，点击 <img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000002349389194.png" width="24" />继续锻炼，点击 <img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000002349229442.png" width="24" /> 结束锻炼。
运动状态下，上下滑动屏幕可查看运动数据。也可右滑进入快捷设置界面，点击<img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000002551443975.png" width="24" />调整锻炼提示音的音量，点击<img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0EX/zh-CN/zh-cn_image_0000002551563967.png" width="24" />进入运动设置界面进行设置。

</TabItem>
