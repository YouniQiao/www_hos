# 减脂塑形

> 本文适用于 **WATCH Ultimate 非凡大师和WATCH GT 6 系列**。

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="ultimate" label="WATCH Ultimate">


结合体重变化、热量缺口、饮食记录、运动等数据，通过设置合理的个性化训练，实现科学减重，保持体重。



## 初始目标设置

1. 在手表应用列表中，点击**减脂塑形**，首次使用会进入用户授权页面，点击**同意**。
2. 根据界面提示，完成设置后进入**减脂塑形**应用首页。


> 💡 **提示：** 如未设置体重信息，请根据界面提示完成相关设置，手表会生成科学减重方案或保持体重方案。 如已在**华为运动健康**应用中设置过当前体重及目标体重，会自动同步至手表侧，手表侧无需重复设置。

</TabItem>
