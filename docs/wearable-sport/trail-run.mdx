# 越野跑

> 本文适用于 **WATCH Ultimate 非凡大师、WATCH GT 6 系列和WATCH 5**。

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

专业运动


|  | 越野跑 |
| --- | --- |


   越野跑

    手表支持专业越野跑功能，可按需进行分段导航并查看全彩地图，数据页面更丰富。


> 💡 **提示：** 使用前需将**华为运动健康**和手表升级到最新版本。越野跑分段导航功能仅部分手表支持，如果您的手表没有分段显示则表示不支持，请以实际界面为准。


## 运动设置


1. 单击手表上键，在应用列表中点击**锻炼**。
2. 上下滑动屏幕选择**越野跑**，点击**<img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0ER/zh-CN/zh-cn_image_0000002364516078.png" width="24" />**图标，进入设置页面。 点击**目标**，可设置距离、时间、总消耗热量的目标值。
3. 点击**提醒**，可设置间隔提醒、心率提醒、训练压力等。运动过程中，满足设置的提醒阈值时，有语音提醒。
4. 点击**数据显示**，设置在运动界面需要显示的数据。 > 💡 **提示：** 可设置的数据显示项有心率、配速、坡度、海拔趋势等，不同的设备有差异，请以实际为准。
5. 可根据需要，设置运动防误触、节拍器、分段等信息。


## 导入越野跑线路

     开始越野跑前，先在**华为运动健康**导入从比赛官网下载的线路文件，并传输到手表。如果导入的是一个新城市的线路，请下载离线地图。
     当导入的越野跑线路包含CP点，海拔数据时，运动中可显示海拔曲线图，以及分段导航数据。


> 💡 **提示：** 海拔曲线图功能目前仅HUAWEI WATCH GT 6 Pro 支持。


1. 在**华为运动健康**，点击**我的 > 我的路线** > **导入路线**，选择本地已下载的路线文件，保存到**我的路线**。
2. 选择目标路线后，点击右上角的**<img src="https://tips-p01-drcn.dbankcdn.cn/hwtips/topic/M0ER/zh-CN/zh-cn_image_0000002373063754.png" width="24" /> > 传输至穿戴设备**，将路线文件导入到手表。


## 开启越野跑


1. 单击手表上键，在应用列表中点击**锻炼 > 越野跑**进入运动准备页面。
2. 上滑屏幕至**更多**页面，点击**导航线路**。
3. 根据需要选择线路，点击**前往运动 > GO图标**，开启越野跑。
4. 上下滑动屏幕，可查看更多运动数据。
