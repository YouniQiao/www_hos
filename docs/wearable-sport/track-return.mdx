# 轨迹返航

> 本文适用于 **WATCH Ultimate 非凡大师和WATCH GT 6 系列**。

手表可进行路线记录及导航，运动开始后或到达目的地后，启动轨迹返航功能，可导航返回至途径地或出发地。


在手表应用列表点击**锻炼**，开始单次户外运动，运动过程中向上滑动屏幕至轨迹页面，可选择**轨迹返航**或**直线返航**返回到运动的途径地或出发地。


如界面无**轨迹返航**、**直线返航**，可点击屏幕使其出现。



> 💡 **提示：** 操场跑步户外运动不支持轨迹界面。 轨迹返航功能仅支持在运动过程中使用，当结束当前户外运动时，将无法使用轨迹返航的功能。
