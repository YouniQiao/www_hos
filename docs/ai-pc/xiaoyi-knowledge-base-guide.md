# 小艺知识库，您的专属知识管家

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/figure/fig_AI_KnowledgeBase.png" alt="小艺知识库" width="720" height="540" />

**小艺知识库**可以协助您高效管理电脑上的本地数据、海量网络信息与知识等，构建个性化知识库。知识库可以为您提供知识收藏、智能分类知识合集、AI 问答等服务，让沉淀积累的知识发挥更大的价值。

![方法](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

1. 单击桌面右下角 ![小艺图标](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/icon/appicon_xiaoyi.png)，打开小艺侧边窗。
2. 单击 ![最大化](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_window_xiaoyi_maxmize.png) 进入小艺应用窗，并单击左侧边栏的**小艺知识库**。
3. 在右侧知识库界面，您可通过以下任一方式，收藏知识并归类：
    * 单击右上角**添加知识**，选择想要添加的内容，如文件、链接、图片等。添加的内容在完成解析后将会自动归类到已有的合集中。
    * 单击已有的合集进入详情界面，然后单击**添加知识**。
    * 单击**创建合集**区域的 ![添加](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_public_add.png) 新建合集，输入合集**名称**、**简介**并开启**智能关联**的开关，然后点击**确认**。在新建的合集界面，单击**添加知识**。
4. 知识添加完成后，可根据需要输入问题，然后点按 Enter 键发送，让**小艺**基于知识库内容进行解答。
