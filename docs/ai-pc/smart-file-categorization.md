# 智能分类，指定文件自动归类

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/figure/fig_fileManager_SmartCollections.png" alt="智能分类" width="720" height="540" />

使用**智慧分类**，将符合共同条件的文件自动归类（如文件类型、文件名、创建日期等），方便快速查看。添加、修改或删除复合条件的文件时，**智慧分类**中的文件列表将自动更新。

![方法](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

1. 单击桌面底部**快捷栏**中的 ![文件管理](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/icon/appicon_filemanager.png)，打开**文件管理**。
2. 单击左边栏的**智慧分类**，在右侧界面空白处右键选择**新建智能分类**，或单击左上角**新建 > 智慧分类**。
3. 在创建智能分类界面，设置分类条件：
   - 在上方输入框中输入关键词，作为分类搜索条件（如文件名、文件内容等）。
   - 单击**分类条件**右侧的 ![展开](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_public_chevron_down.png) 展开，单击**新增**后在下拉选项中选择搜索属性。如有需要，可单击**新增**添加多个条件。
4. 设置完成后，输入**智能分类名称**，然后单击**创建智能分类**。
