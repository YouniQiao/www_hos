# 文档助理，快速总结摘要和问答

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/figure/fig_AI_DocumentAssistant.png" alt="文档助理" width="720" height="540" />

使用**文档助理**，帮您快速理解各类文档进行总结和问答，还能提取核心观点、生成思维导图，助您高效阅读。

![方法](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

1. 单击桌面右下角 ![小艺](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/icon/appicon_xiaoyi.png)，打开小艺侧边窗。
2. 单击 ![最大化](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_window_xiaoyi_maxmize.png) 进入小艺应用窗，并单击左侧边栏的**文档助理**。
3. 单击**选择文档**或将文档拖拽至解析区域，**小艺**将自动解析文档内容，生成摘要总结。您也可以单击**输入网址**，输入待解析的网页地址（仅支持部分网址）。
4. 输入问题然后点按 Enter 键发送，让**小艺**基于文档内容进行解答；或者单击右侧界面菜单执行更多操作，如脑图、翻译、精读等。
