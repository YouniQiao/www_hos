# 智能推荐，关联文档一键直达

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/figure/fig_fileManager_SmartRecommend.png" alt="智能推荐" width="720" height="540" />

系统可对**文件管理**中的本地文件进行智能分析，推荐关联文档，助您快速查阅。

![方法](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

1. 单击桌面底部**快捷栏**中的 ![文件管理](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/icon/appicon_filemanager.png)，打开**文件管理**。
2. 单击选中待查看的文档，然后单击右上角**推荐**，系统将根据**内容相近**或**名称相近**为您推荐相关文档。
3. 您可在右侧推荐文档列表中双击打开任一文档进行查看，或右键选择文档执行更多操作（如打开所在文件夹、选择打开方式、添加到智能分类等）。

![提示](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_tips.png) 提示

仅电脑本机的部分文件格式支持智能推荐（不支持云盘、网络共享、外接设备等路径文件），请以实际情况为准。
