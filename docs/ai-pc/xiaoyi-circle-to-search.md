# 小艺圈选，快速提取搜索

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/figure/fig_AI_CeliaCircle.png" alt="小艺圈选" width="720" height="540" />

遇到感兴趣的屏幕内容，您可以在屏幕上圈选，让小艺识图搜百科或找同款。

![方法](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

1. 在查看图片时，单击桌面右下角 ![小艺图标](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/icon/appicon_xiaoyi.png) 打开小艺侧边窗，单击圈选图标 ![圈选图标](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_celia_AI_screenshot.png)。您也可以点按（键盘上的）**鸿蒙键** ![鸿蒙键](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/keyboard/HM_keyboard_OH_1.png) + **Alt** 组合键快速进入圈选模式。
2. 通过鼠标或单指触屏操作，在屏幕画面中圈出一个封闭图形，将想要的内容圈出来。当圈选范围为较完整的物体时，**小艺**将自动感应截图内容并推荐对应的服务，如景点介绍、找同款、文字提取等。
3. 您可以根据需要，单击对应的服务，或单击**问问小艺**。


![提示](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_tips.png) 提示

**问问小艺**会根据识别内容出现不同选项，请按实际操作。
