# 控制中心个性编辑

<video controls src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/video/vid_ControlCenter_edit.mp4" width="720" height="540"></video>

您可根据需要编辑**控制中心**，如调整快捷开关位置、隐藏不常用的快捷开关、添加快捷开关等，帮助您更好的使用控制中心。

![方法](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

1. 单击桌面右下角**状态栏**中间区域（包含电池图标 ![电池](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/status/HM_public_battery.png)）或点按（键盘上）**鸿蒙键** ![鸿蒙键](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/keyboard/HM_keyboard_OH_1.png) + **C** 组合键，打开**控制中心**。
2. 单击 ![编辑](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_public_edit_pencil.png) 进入编辑界面，您可进行如下操作：
    *   **调整快捷开关位置：** 光标移至待移动的快捷开关区域，单指长按触控板（或鼠标左键）并移动，将其拖拽至想要的位置（控制中心编辑界面白色边框内），然后松开手指。
    *   **隐藏快捷开关：** 单击快捷开关上的 ![移除](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/ic_public_remove_red.png)，将其隐藏。
    *   **添加快捷开关：** 单击下方隐藏的快捷开关上的 ![添加](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_public_add_norm_green.png)，将其添加到**控制中心**显示。
3. 编辑完成后，单击 ![确定](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_public_ok.png) 保存并自动返回**控制中心**。
