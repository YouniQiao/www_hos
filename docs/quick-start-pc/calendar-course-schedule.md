# 日历课程表，随时查看课程安排

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/figure/fig_calendar_CourseSchedule.png" alt="日历课程表" width="720" height="540" />

在日历中，您可通过图片导入的方式快捷创建课程表，还可将课程表卡片添加至桌面，轻松管理和查看课程。

![方法](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

1. 单击桌面底部**快捷栏**中的 ![应用中心](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/icon/appicon_allapps.png) 进入**应用中心**，单击打开**日历**。
2. 单击 **添加 > 添加课表 > 相册导入 > 开始相册导入**，选择课程表图片后单击**完成**，快速导入课程表。
3. 完成解析后，检查并修改导入结果，然后单击**下一步**。
4. 根据需要调整课程表名称、开学和期末时间、课表时间等，单击 ![确定](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_public_ok.png) 完成创建。
5. 根据需要，在电脑桌面右键选择**添加卡片**，单击**日历**然后在右侧界面向下滑动到课程表卡片，单击卡片添加至桌面。

![提示](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_tips.png) 提示

已创建的课程表，若某课程信息有误，可单击该课程所在位置，然后单击**编辑**重新修改。
