# 涂鸦创作，还能一键还原

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/figure/fig_gallery_graffiti.png" alt="涂鸦创作" width="720" height="540" />

您可以在图片上，用各种涂鸦笔触，轻松画出标准图形。保存图片后，还能一键恢复原图。

![方法](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/fig_method.png) 方法

1. 单击桌面底部**快捷栏**中的 ![图库](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/icon/appicon_gallery.png)，打开**图库**。您也可以单击桌面底部 ![应用中心](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/icon/appicon_allapps.png) 进入**应用中心**，单击打开**图库**。
2. 双击待编辑的图片进入详情界面，然后单击 ![编辑](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_public_edit_pencil.png) > ![涂鸦](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_gallery_edit_penkit.png) 进入涂鸦编辑界面，您可执行以下操作：
   - **添加涂鸦：** 选择不同笔触和颜色，用手指操作屏幕进行创作。如想快速画出标准图形，可单击 ![更多](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_public_more_1.png)，开启**抬笔成形**的开关，然后用手指在屏幕上的图片区域一笔画完图形后停顿 1 s，图形即可变成标准图形（如圆形、心形、三角形等）。
   - **添加马赛克：** 单击 ![马赛克](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_gallery_edit_mosaic.png)，可为图片添加马赛克。
   - **擦除画笔内容：** 单击 ![橡皮擦](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_gallery_edit_rubber.png) 选择橡皮擦，可擦除涂鸦或马赛克内容。再次单击 ![橡皮擦](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_gallery_edit_rubber.png)，可在弹出框中选择擦除方式（如**点擦除**或**笔画擦除**）、**擦除直径**或**仅擦除荧光笔**。
3. 编辑完成后，单击 ![确认](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_public_ok.png) > ![保存](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_public_save.png)，可选择另存为或覆盖原图保存。

图片保存后，如需修改重做，可双击图片进入详情界面，然后单击 ![编辑](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/HM_public_edit_pencil.png) > ![重置](https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031BETA/resource/card/202511200rXkw5/zh-cn/image/common/buttons/ic_public_reset.png) 恢复为原图后再次编辑。
