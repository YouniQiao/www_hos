# 单应用音量调节

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202602261wvwjj/zh-cn/image/figure/fig_ControlCenter_volume.png" width="720" height=""/>

您可对应用的播放音量单独进行调节，享受纯净音效体验的同时，避免其他杂音影响。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202602261wvwjj/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1. 打开音乐、视频等应用时，单击桌面右下角状态栏中间区域（包含电池图标）或点按（键盘上）鸿蒙键 + C 组合键，打开控制中心。

2. 光标移动至音量条区域，单击右侧出现的箭头，进入声音详情界面。

3. 在声音界面上方，拖动系统声音或应用的音量滑块，调节播放音量。

4. 声音界面默认显示最多 3 个最近使用的应用音量滑块，如需调节更多应用的播放音量，可单击下方声音设置打开设置，然后在右侧界面单击应用声音管理进入详情界面进行调节。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202602261wvwjj/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

+ 使用该功能前，请将您的设备升级到最新版本。

+ 应用的播放音量大小受限于系统音量，建议您先将系统声音调整至合适音量，再对应用的音量进行单独调节。
