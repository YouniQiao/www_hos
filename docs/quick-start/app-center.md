# 应用中心，高效管理应用

<iframe src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/2025102603wmgm/zh-cn/image/video/10044758_f014_appcentre.mp4#toolbar=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="360" height="486"> </iframe>

桌面应用太多导致杂乱、难查找？将不常用的应用从桌面移除，精简桌面告别杂乱，需要时可一步进入应用中心查找使用。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

进入设置 > 桌面和个性化 > 桌面设置 > 应用中心，打开应用中心开关。

**精简桌面应用**

长按桌面上的应用，点击移除 > 从桌面移除。移除后，可进入应用中心查找使用或重新添加至桌面。

**使用应用中心**

在桌面向上滑动，进入应用中心：

+   搜索或查找应用，可直接打开使用。
    
+   长按应用并拖拽至桌面，可将其添加至桌面。
    

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

+   使用该功能前，请将您的设备升级到最新版本。
    
+   若选择卸载应用，则将从桌面和应用中心同步卸载，无法在应用中心重新查找使用。
