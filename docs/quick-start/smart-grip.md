# 智感握姿，单手操作更顺手

<iframe src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202612220zjZV2/zh-cn/image/video/10044835_f009_reachability.mp4#toolbar=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="360" height="486"> </iframe>

智感握姿可智能识别握持手势，按键跟手而动，自动调整到顺手的一侧，单手也能轻松操作。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202612220zjZV2/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

以接听来电为例：

1. 进入电话 > <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202612220zjZV2/zh-cn/image/common/buttons/ic_AITouchmore.png" width="24" height="24"/> > 设置 > 解锁后来电通知，选择横幅。

2. 进入设置 > 系统 > 智感握姿，打开智感握姿开关。

3. 来电时，当您拿起手机，屏幕侧边将出现 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202612220zjZV2/zh-cn/image/common/buttons/ic_call.png" width="24" height="24"/> 和 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202612220zjZV2/zh-cn/image/common/buttons/ic_end_call.png" width="24" height="24"/> 按键，方便单手操作：
   - 右手握手机，按键在屏幕右侧。
   - 左手握手机，按键在屏幕左侧。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202612220zjZV2/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

+ 当前智感握姿支持电话、畅连通话、部分语音及视频通话应用，请解锁进入桌面使用。
