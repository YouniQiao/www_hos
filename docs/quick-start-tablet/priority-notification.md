# 优先通知，重要通知不错过

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202606110qRije/zh-cn/image/figure/10044804_f012_prioritynotification.png" width="560" height=""/>

通知太多，重要消息容易被淹没？优先通知可自定义和智能识别重要消息，如预约提醒、缴费还款、重要待办等，在锁屏和通知中心优先置顶显示，一目了然。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202606110qRije/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1. 进入设置 > 通知和状态栏 > 优先通知，进入所需应用开启优先通知。

2. 按需选择优先显示的方式，例如设为自定义和智能识别。当您选择带"自定义"的方式时，可添加关键词，标题包含关键词的通知将优先显示。

3. 从屏幕顶部左侧下滑出通知中心，重要通知将带有"优先"标识，从上至下置顶显示。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202606110qRije/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

+ 若在优先通知中未找到所需应用，请进入设置 > 通知和状态栏，打开应用的右侧开关。
