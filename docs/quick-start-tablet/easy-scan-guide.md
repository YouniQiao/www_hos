# 扫一扫，无惧远距离、破损码

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/figure/20013791_f001_Scan.png" alt="扫一扫" width="560" height="" />

使用扫一扫功能，扫码前无需打开应用，无惧破损码、反光码、变形码等复杂扫码场景，一步即扫，直达服务。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

从屏幕顶部右侧下滑出**控制中心**，点击 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_scan.png" width="24" height="24"/>，选择**扫一扫**，调整平板位置，使二维码显示在取景框中。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

目前已支持扫码支付、点餐、骑行、停车缴费、取快递、车辆充电等多种场景。
