# 实时调整手写字迹

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/figure/10044681_f016_Handwriting.png" alt="实时调整手写字迹" width="560" height="" />

手写输入文字时，可将字迹调整为楷体或行书等字体。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

1. 在**笔记**中打开一个正在编辑的笔记，手写输入一段文字。
2. 点击 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_lasso.png" width="24" height="24"/>，圈选需要调整的文字，选择**字迹调整**。
3. 选择字体后，平板自动将圈选的文字调整为所选字体。
