# 编辑控制中心

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/figure/10099735_f005_EditSwitches.png" alt="编辑控制中心" width="560" height="" />

根据使用习惯，将常用快捷开关移动到顺手的位置，还可调整亮度条、音量条的布局，打造专属控制中心。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B030/resource/card/202512281uswxk/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

从屏幕顶部右侧下滑出**控制中心**，点击左上角 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_controlpanel_edit.png" width="24" height="24"/>，进入编辑界面：

- **移除开关**：点击快捷开关上的 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_remove.png" width="24" height="24"/> 将其移除出面板。
- **添加开关**：点击 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_quickswitch_add.png" width="24" height="24"/>，进入添加界面，点击快捷开关上的 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_quickswitch_add1.png" width="24" height="24"/> 将其添加至面板，然后点击上方灰色区域退出添加界面。
- **移动位置**：长按并拖动快捷开关，调整其在面板中的位置。

编辑完成后，点击 <img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/common/buttons/ic_choose_done.png" width="24" height="24"/> 保存设置。
