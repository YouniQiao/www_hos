# 设备丢失，关机也能找

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/DOC/C00B031/resource/card/202608192tbkfh/zh-cn/image/figure/10044536_f001_FindPhone.png" alt="设备丢失，关机也能找" width="560" height="" />

开启查找设备功能后，可对华为的手机、平板等设备发起定位查找，关机状态也能定位，保护您的设备和数据安全。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/EMUI/C00B030/resource/card/202503041becsx/zh-cn/image/common/buttons/fig_method.png" width="24" height="24"/> 方法

**在平板上开启查找设备并完成安全验证：**

1. 进入**设置**，登录**华为账号**进入云空间，点击**查找设备**。
2. 开启**查找我的平板**开关。
3. 根据界面提示操作，完成安全认证。

**通过其他设备定位丢失的平板：**

1. 在其他华为设备上打开**查找设备**应用。
2. 登录与丢失设备相同的**华为账号**。
3. 点击需查找的设备，可进行定位、远程锁定或擦除数据等操作。

完成安全认证且满足版本条件的平板在关机状态下也能被定位查找。

<img src="https://tips-p01-drcn.dbankcdn.cn/MODEL/EMUI/C00B030/resource/card/202508300vZjQz/zh-cn/image/common/buttons/fig_tips.png" width="24" height="24"/> 提示

平板未连接网络或已关机时，处于离线状态，您可设置上线通知，待平板上线后将发送通知给您。
